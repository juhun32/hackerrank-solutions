## Task
The provided code stub reads and integer, n, from STDIN. For all non-negative integers i < n, print i^2.

---

## Input Format
The first and only line contains the integer, n.

---

## Output Format
Print n lines, one corresponding to each i.

---

## Sample Input 0
<pre>
5
</pre>
---

## Sample Output 0
<pre>
0
1
4
9
16
</pre>
