## Task
You are given a string. Split the string on a " " (space) delimiter and join using a - hyphen.

---

## Input Format
The one line contains a string consisting of space separated words.

---

## Sample Input 0
<pre>
this is a string   
</pre>
---

## Sample Output 0
<pre>
this-is-a-string
</pre>
